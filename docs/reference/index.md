# Reference

Use these pages when wiring integrations or checking exact option names:

- [HTTP API Reference](./http-api.md): public ingest endpoints, payloads, responses, errors, and compression support.
- [Generated SDK and React Reference](./sdk/README.md): generated from TypeScript source comments with TypeDoc.

The generated SDK reference is rebuilt with:

```bash
npm run docs:reference
```

Run that command after changing exported SDK or React types.
